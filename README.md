# Aplicación de Simón Dice
-Memory Quest
Memory Quest es un juego interactivo de memoria en el que el jugador debe emparejar cartas idénticas en el menor tiempo posible. A medida que se avanza de nivel, la dificultad aumenta y se pone a prueba la agilidad mental, la concentración y la memoria visual del jugador.

-Características principales
Temporizador: cada partida cuenta con un tiempo límite.

Vidas: el jugador dispone de un número limitado de intentos.

Niveles: la dificultad incrementa progresivamente (más cartas por nivel).

Interfaz limpia y moderna: diseñada con una estética minimalista en tonos oscuros.

Sistema de puntuación: los puntos se obtienen según el tiempo y la precisión del jugador.
